# KillPort v1.0.0

KillPort is a simple tool for windows to kill processes that are using specific ports, I made it, so I don't need 
to use bash files or keep doing all by hand all the time.

### Requirements
dotnet 8 runtime or higher

### Installation
Just use the zip on the releases page and set your system variables to use it. No big deal.

### How to use
On your terminal, just type: ``KillPort <port>``
